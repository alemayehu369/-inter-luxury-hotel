# Inter Luxury Hotel

A modern, responsive, full-stack hotel management and booking platform built with modern web technologies.

## Features

- User Authentication
- Hotel Room Booking
- Restaurant Ordering
- Event Hall Booking
- Online Payments
- Customer Dashboard
- Admin Dashboard
- Room Management
- Restaurant Menu Management
- Booking Management
- Reports & Analytics
- Responsive Design

## Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cloudinary
- Stripe/Chapa Payment
